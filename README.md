# Residual Theory Playground

This is smallish playground to visually present ideas 
by Barry O`Reilly in his talk ["An Introduction to Residuality Theory - NDC Oslo 2023"](https://www.youtube.com/watch?v=0wcUG2EV-7E) 

Also I should give a praise to Oskar Dudycz for his article [Residuality Theory: A Rebellious Take on Building Systems That Actually Survive](https://www.architecture-weekly.com/p/residuality-theory-a-rebellious-take)
which also gives some examples, which can be shown in this playground.
